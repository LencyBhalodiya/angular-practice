what is signals
A signal is a special kind of variable that can notify other parts of your application when its value changes.
Signals are used to manage state and reactivity in a more efficient way compared to traditional methods like
Observables or EventEmitters.
Signals are designed to be simple and easy to use, making it easier to build reactive applications.
set method is used to set the value of a signal to a new value.
update method is used to update the value of a signal based on its current or previous value.
Signals return a function that can be called to get the current value of the signal.
